# `eth_estimateGas`

{m.description}

:::note

For more accurate gas estimation on Linea, use [`linea_estimateGas`](linea-estimategas) instead. It
returns the gas limit, base fee per gas, and priority fee per gas in a single call, accounting for
Linea's L1 data-submission costs.

:::

## Parameters

- `transaction`: _\[required]_ Transaction call object (same fields as [`eth_call`](eth-call)).
- `blockParameter`: _\[optional]_ Hexadecimal block number, or `latest`.
