# `eth_newFilter`

{m.description}

## Parameters

- `filter`: _\[required]_ Filter object (same shape as [`eth_getLogs`](eth-getlogs)):
  - `fromBlock`: _\[optional]_ Hexadecimal block number, or `latest`, `earliest`. Defaults to `latest`.
  - `toBlock`: _\[optional]_ Hexadecimal block number, or `latest`, `earliest`. Defaults to `latest`.
  - `address`: _\[optional]_ Contract address or array of addresses.
  - `topics`: _\[optional]_ Array of 32-byte topic values. Each position can be a single value, an array of values (OR), or `null` (wildcard).

## Returns

{m.returns}

## Example

### Request

```bash
curl https://rpc.linea.build \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_newFilter",
    "params": [
      {
        "address": "0x176211869ca2b568f2a7d4ee941e073a821ee1ff",
        "topics": [
          "0xddf252ad1be2c89b69c2b068fc378daa952ba7f163c4a11628f55a4df523b3ef"
        ]
      }
    ],
    "id": 1
  }'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x10ff687202f305efab..."
}
```
