# `web3_clientVersion`

Returns the client software version running the node.

## Parameters

None.

## Returns

String describing the client and its version.

## Example

<RpcPlayground
  method="web3_clientVersion"
  params={[]}
  exampleResponse={{
    jsonrpc: "2.0",
    id: 1,
    result: "Geth/v1.17.2-stable-be4dc0c4/linux-amd64/go1.26.1"
  }}
/>
