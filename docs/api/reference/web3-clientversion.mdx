# `web3_clientVersion`

{m.description}

## Parameters

None.

## Returns

{m.returns}

## Example

### Request

```bash
curl https://rpc.linea.build \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"web3_clientVersion","params":[],"id":1}'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "Geth/v1.17.2-stable-be4dc0c4/linux-amd64/go1.26.1"
}
```
