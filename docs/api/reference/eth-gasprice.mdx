# `eth_gasPrice`

Returns the current gas price in wei.

## Parameters

None.

## Returns

Hexadecimal value of the current gas price in wei.

## Example

<RpcPlayground
  method="eth_gasPrice"
  params={[]}
  exampleResponse={{
    jsonrpc: "2.0",
    id: 1,
    result: "0x27f6780"
  }}
/>
