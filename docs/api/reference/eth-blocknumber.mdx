# `eth_blockNumber`

Returns the number of the most recent block.

## Parameters

None.

## Returns

Hexadecimal integer of the current block number.

## Example

<RpcPlayground
  method="eth_blockNumber"
  params={[]}
  exampleResponse={{
    jsonrpc: "2.0",
    id: 1,
    result: "0x1ce5452"
  }}
/>
