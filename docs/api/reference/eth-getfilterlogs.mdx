# `eth_getFilterLogs`

{m.description}

## Parameters

- `filterId`: _\[required]_ The filter ID returned by [`eth_newFilter`](eth-newfilter).

## Returns

{m.returns}

## Example

### Request

```bash
curl https://rpc.linea.build \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_getFilterLogs",
    "params": ["0x10ff687202f305efab..."],
    "id": 1
  }'
```
