# `eth_getTransactionByHash`

Returns transaction data for a given transaction hash.

## Parameters

- `transactionHash`: _\[required]_ 32-byte transaction hash.

## Returns

A transaction object, or `null` if no transaction is found.

## Example

### Request

```bash
curl https://rpc.linea.build \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_getTransactionByHash",
    "params": [
      "0x972fd73b83ee0d567143ebc67509d4fc98fa1b1941c2c145a342d3d10715c259"
    ],
    "id": 1
  }'
```
