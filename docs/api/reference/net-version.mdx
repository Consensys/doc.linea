# `net_version`

Returns the network ID as a decimal string. On Linea Mainnet this is `59144`. On Linea Sepolia it
is `59141`.

## Parameters

None.

## Returns

Decimal string of the network ID.

## Example

<RpcPlayground
  method="net_version"
  params={[]}
  exampleResponse={{
    jsonrpc: "2.0",
    id: 1,
    result: "59144"
  }}
/>
