# `eth_chainId`

Returns the chain ID of the connected network. Linea Mainnet is `0xe708` (59144). Linea Sepolia is
`0xe705` (59141).

## Parameters

None.

## Returns

Hexadecimal value of the chain ID.

## Example

<RpcPlayground
  method="eth_chainId"
  params={[]}
  exampleResponse={{
    jsonrpc: "2.0",
    id: 1,
    result: "0xe708"
  }}
/>
