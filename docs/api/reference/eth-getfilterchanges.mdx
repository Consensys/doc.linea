# `eth_getFilterChanges`

{m.description}

## Parameters

- `filterId`: _\[required]_ The filter ID returned by [`eth_newFilter`](eth-newfilter) or [`eth_newBlockFilter`](eth-newblockfilter).

## Returns

{m.returns}

## Example

### Request

```bash
curl https://rpc.linea.build \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_getFilterChanges",
    "params": ["0x10ff687202f305efab..."],
    "id": 1
  }'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": []
}
```
