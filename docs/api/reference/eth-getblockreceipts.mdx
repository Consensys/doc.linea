# `eth_getBlockReceipts`

Returns all transaction receipts for a given block.

## Parameters

- `blockParameter`: _\[required]_ Hexadecimal block number, or `latest`, `earliest`, `pending`, `finalized`.

## Returns

Array of transaction receipt objects for the block. Each receipt has the same shape as the response
from [`eth_getTransactionReceipt`](eth-gettransactionreceipt).
