# `eth_maxPriorityFeePerGas`

Returns the current priority fee per gas in wei. This is the tip paid to validators above the base
fee.

## Parameters

None.

## Returns

Hexadecimal value of the current max priority fee per gas in wei.

## Example

### Request

```bash
curl https://rpc.linea.build \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"eth_maxPriorityFeePerGas","params":[],"id":1}'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x396c8ee"
}
```
