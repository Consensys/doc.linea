# `net_listening`

Returns `true` if the node is actively listening for network connections.

## Parameters

None.

## Returns

Boolean. `true` if the node is listening, `false` otherwise.

<RpcPlayground
  method="net_listening"
  params={[]}
  exampleResponse={{
    jsonrpc: "2.0",
    id: 1,
    result: true
  }}
/>
