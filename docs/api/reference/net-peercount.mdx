# `net_peerCount`

Returns the number of peers currently connected to the node.

## Parameters

None.

## Returns

Hexadecimal integer of the peer count.

## Example

<RpcPlayground
  method="net_peerCount"
  params={[]}
  exampleResponse={{
    jsonrpc: "2.0",
    id: 1,
    result: "0x64"
  }}
/>
