# `net_peerCount`

Returns the number of peers currently connected to the node.

## Parameters

None.

## Returns

Hexadecimal integer of the peer count.

## Example

### Request

```bash
curl https://rpc.linea.build \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"net_peerCount","params":[],"id":1}'
```

### Response

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x64"
}
```
