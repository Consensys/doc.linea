## PauseManager

### PAUSE_ALL_ROLE

```solidity
bytes32 PAUSE_ALL_ROLE
```

This is used to pause all pausable functions.

### UNPAUSE_ALL_ROLE

```solidity
bytes32 UNPAUSE_ALL_ROLE
```

This is used to unpause all unpausable functions.
