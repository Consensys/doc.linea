Coordinator configuration controls batching, blob creation, and finalization behavior. Configuration
includes:

- Conflation deadlines
- Batch and blob size limits
- Proof generation coordination
- Finalization layer connections
- File system paths for batch data

For information about the coordinator's role, see [Coordinator](/protocol/coordinator).
