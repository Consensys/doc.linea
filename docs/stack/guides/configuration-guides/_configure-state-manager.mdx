State manager configuration controls state representation and proof generation data. Configuration
includes:

- Besu node with Shomei plugin settings
- Shomei node configuration
- State snapshot settings
- Merkle proof generation configuration

For information about the state manager's role, see [State Manager](/protocol/state-manager).
