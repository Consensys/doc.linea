Maru configuration is managed via `maru-config.toml`. Common configuration areas include:

- QBFT consensus settings
- P2P networking
- Engine API connection to execution layer
- Observability and metrics

For basic deployment configuration, see [Deploy Maru](/stack/guides/deployment-guides/deploy-maru).
