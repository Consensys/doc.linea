Linea Mainnet is the public, permissionless zkEVM layer 2 network built on
[Lineth](../../protocol/linea-vs-lineth.mdx). It processes Ethereum-compatible transactions,
generates zero-knowledge proofs, and finalizes state to Ethereum.

For protocol mechanics, see [how the Lineth protocol works](../../protocol/overview.mdx).

Linea is a secure, low-cost, and Ethereum-equivalent layer 2 blockchain built to bring the world
onchain while inheriting security from Ethereum through its rollup design.

Linea has been incubated within Consensys. Our mission is to empower developers to build the next generation of decentralized applications and onboard the next billion users to web3.

## Security for all

Built as a layer above Ethereum, Linea inherits security from the Ethereum network while increasing
throughput and reducing transaction fees.

→ Learn about [Linea's architecture](../../protocol/architecture/index.mdx)

## Advanced zero-knowledge proof technology

By leveraging zero-knowledge (zk) proofs, Linea removes the seven-day lockup period associated with
fraud proofs in other layer 2 solutions built according to the optimistic rollup model. Withdrawal
timing and cost still depend on proof generation, finalization layer conditions, bridge
configuration, and network fees.

→ Learn about how proofs fit within [Linea's transaction lifecycle](../../protocol/architecture/index.mdx#transaction-lifecycle)

## True Ethereum equivalence

Linea supports the Ethereum Virtual Machine (EVM), so Ethereum-based projects and assets can be
deployed on and bridged to Linea with minimal changes.

→ Learn about the small differences between building on [Ethereum and Linea](./ethereum-differences.mdx)

## Fast finality

Soft finality within 1s because Linea has short block times and does not reorg confirmed L2 transactions.

→ Learn about Linea's [finality lifecycle](./transaction-finality.mdx)

## Predictable pricing

Clarity over Linea prices for transactions, how to estimate fees, and how Layer 1 and Layer 2 costs are accounted for.

→ Learn about Linea's [pricing strategy](./predictable-pricing.mdx)

## Recoverable state

Dedicated plugin to reconstruct Linea's state as it was at the last point in time data was finalized.

→ Learn about [recovering state](./recoverable-state.mdx)
