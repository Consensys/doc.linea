Linea's principal testnet is now Linea Sepolia; please use it for any testnet purposes.

Linea Goerli was officially decommissioned on May 31 2024.

## Running a Sepolia node

Linea Goerli nodes should no longer work; however, if you still run one, please transition to a 
Linea Sepolia node as soon as possible.

You will need to spin up a new node. The method is the same as usual, and as detailed in our 
[guide](developers/guides/run-a-node). However, you must use new genesis files specific to Linea 
Sepolia:

- [Linea Sepolia Geth genesis file](pathname:///files/sepolia/genesis-sepolia-geth.json)
- [Linea Sepolia Besu genesis file](pathname:///files/sepolia/genesis-sepolia-besu.json)

Here are the bootnodes you should use when initiating your client:

## Updating your dapps

For your dapp to remain functional, you must:

- Deploy any smart contracts anew on Linea Sepolia
- Ensure your dapps reference the new smart contract deployments.

## Linea Sepolia network details

You can find a more thorough overview of key Linea Sepolia details on our [network info page](/developers/quickstart/info-contracts). 

Here are the key details:

- Chain ID: 59141
- RPC URL: https://rpc.sepolia.linea.build
- Block explorer: 
    - [Lineascan](https://sepolia.lineascan.build)
    - [Blockscout](https://explorer.sepolia.linea.build)

### MetaMask integration

Linea Sepolia has been added as a default testnet in MetaMask in the following releases:

- MetaMask Extension: 11.13.1 — rolled out from March 29 2024.
- MetaMask Mobile: 7.19.0 — rolled out from April 1 2024.

Linea Goerli was removed from MetaMask in Extension v11.17.0 and MetaMask Mobile v7.23.
